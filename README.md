# Taleem360-AI Learning Ecosystem

[![Branded with SaaSSkul](https://img.shields.io/badge/Branded%20with-SaaSSkul-blue?style=for-the-badge)](https://saasskul.com)

**Taleem360-AI Learning** is a production-level Node.js and Express application designed to empower AI enthusiasts across Pakistan. It combines a robust School ERP foundation with a cutting-edge AI-driven curriculum to help students, freelancers, and community leaders scale from beginners to sustainable, self-funded projects.

## 🚀 Key Features
- **Smart School ERP**: Comprehensive student management, fee tracking, and academic grading system.
- **AI Learning Path**: Curriculum aligned with Pakistan's national framework (10th Grade AI).
- **Economic Empowerment**: Modules focused on AI-powered freelancing and monetization.
- **Community Focus**: Tools and guides for community leaders to solve local challenges via AI.
- **Modern Tech Stack**: Built with Node.js, Express, EJS, and MongoDB for high performance and scalability.

## 🛠 Tech Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Frontend**: EJS (Embedded JavaScript), Vanilla CSS3 (Custom Design System)
- **Auth**: Passport.js (Google OAuth 2.0)
- **Security**: Helmet, Express-Session, MongoStore

## 🏁 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (Local or Atlas)
- Google OAuth Credentials

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/FazalShahidLatif/taleem360-ai-learning.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root and add your configuration:
   ```env
   PORT=3000
   MONGODB_URI=your_mongodb_uri
   SESSION_SECRET=your_secret
   GOOGLE_CLIENT_ID=your_id
   GOOGLE_CLIENT_SECRET=your_secret
   ADMIN_EMAIL=your_admin_email
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
